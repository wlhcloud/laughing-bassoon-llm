# 文保单位信息抽取与验证系统

## 项目简介：

本项目用于批量处理哈尔滨市各区县的文物保护单位公布文件，通过 Excel 解析、文件相似度匹配、OCR 内容提取、大模型结构化抽取 等核心流程，自动从公布文件（Word/PDF 格式）中抽取文保单位的名称、详细地址等关键信息，并与 Excel 中的文保单位名录进行双重比对验证，确保抽取信息的准确性和完整性。

项目核心价值的是替代人工逐份查阅、录入的繁琐流程，实现多区县、多文件的批量自动化处理，提升文保单位信息整理的效率和准确率，为文保单位信息归档、管理提供标准化的数据支撑。

## 核心特性

🔍 智能匹配：基于模糊相似度算法，实现「通知名称与公布文件」「文保单位名称与 OCR 内容」的双重精准匹配

📄 多格式兼容：支持 Word（.docx）、PDF 格式公布文件的递归遍历与 OCR 解析，适配复杂目录结构

🤖 结构化抽取：调用大模型（Qwen-32B）实现文保单位名称、详细地址的结构化输出，避免手动整理

📊 批量处理：自动遍历哈尔滨市所有区县，一键完成全量数据的抽取与验证，支持日志追溯

✅ 双重校验：通过「文件匹配阈值」「名称匹配阈值」双重校验，确保抽取结果的准确性
